# Dino Project

Опис проєкту

Dino Project — це гра, створена на Python з використанням бібліотеки Pygame за мотивами відомої гри Chrome Dino. Гравець керує динозавром, який повинен уникати перешкод та набирати якомога більше балів.

У проєкті реалізовано систему зміни дня і ночі, різні типи об'єктів навколишнього середовища та збільшену кількість кактусів для підвищення складності проходження.

Можливості

* Керування персонажем за допомогою клавіатури.
* Генерація кактусів як основних перешкод.
* Система зміни дня та ночі.
* Хмари та зорі для створення динамічного фону.
* Підрахунок результату під час гри.
* Зростання складності в процесі проходження.

Принцип роботи

Після запуску гри відкривається ігрове вікно. Динозавр автоматично рухається вперед, а на його шляху з'являються кактуси. Гравець повинен вчасно виконувати стрибки, щоб уникнути зіткнень. З часом складність гри збільшується завдяки більшій кількості перешкод. Також у грі реалізовано зміну дня і ночі та рухомі елементи фону.

При зіткненні динозавра з кактусом гра завершується.

Використані технології

* Python
* Pygame

Встановлення та запуск

1. Клонувати репозиторій:

git clone https://github.com/shvedmaksym/dino_project.git

2. Перейти до папки проєкту:

cd dino_project

3. Встановити бібліотеку Pygame:

pip install pygame

4. Запустити гру:

python main.py


Структура проєкту

dino_project/
├── main.py       # головний файл запуску гри
├── player.py     # логіка персонажа
├── cactus.py     # логіка перешкод
├── cloud.py      # створення та рух хмар
├── star.py       # створення зірок
├── settings.py   # налаштування гри
└── .gitignore
